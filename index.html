How does it work inside?

In short: JSON ‚Üí SQLite FTS ‚Üí Avalonia UI. Everything lives on your local disk.

Seforim corpus as JSON

Each sefer is stored as a clear, inspectable JSON file, easy to version, back up, and manage in Git. The app slices it into dapim / simanim and prepares entries for indexing.

SQLite FTS index

SQLite‚Äôs Full-Text Search engine builds an inverted index over the text, so textual queries return results almost instantly ‚Äî even on a modest laptop, completely offline.

Avalonia UI

Avalonia provides a modern, cross-platform UI: text reader, shared toolbar, search panels, bookmarks, and a comfortable dark mode for long learning sessions.
