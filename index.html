Why TaryagNotes?

How it works

Under the hood, TaryagNotes bridges a simple JSON corpus with a serious search engine.

JSON corpus

Seforim are stored as JSON on disk: easy to version, inspect, and transform. The app reads them and splits into daf / siman units for indexing.

SQLite FTS index

A local SQLite FTS database builds inverted indexes for the seforim, enabling Bar-Ilan-style, prefix and proximity search in milliseconds.

Avalonia front-end

Avalonia provides a modern cross-platform desktop UI: rich e-reader, split views, bookmarks, and a global toolbar shared across the ‚Äúmitzvos‚Äù and ‚Äúseforim‚Äù worlds.
